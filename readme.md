## 🧲 Simulador Interactivo de la Ley de Lenz
Aplicación web desarrollada con Streamlit que permite simular y visualizar la inducción electromagnética en una bobina, mostrando cómo la FEM se opone al cambio del flujo magnético.

## Funcionalidades
- Simulación de campo magnético (movimiento y senoidal)
- Visualización de:
  - Flujo magnético Φ(t)
  - FEM inducida
  - Relación de fase
- Parámetros ajustables:
  - Número de espiras (N)
  - Área
  - Ángulo θ
  - Frecuencia y amplitud
- Exportación de datos en CSV
